In the context of the rapid growth of e-commerce and digital transformation, online shopping has become an essential part of modern consumer behavior. Bicycles and related accessories, traditionally sold through physical stores, are increasingly shifting toward online business models to reach a wider audience, especially among young people and sports enthusiasts.

This project focuses on the development of **"Bicycle Online Shop"** â€“ a fully functional web-based e-commerce system specialized in selling bicycles and accessories. Notably, the system integrates:
- A **24/7 chatbot** for automated customer support  
- A **comprehensive admin management panel** for managing products, orders, and users  

Built using a **three-layer architecture** (Presentation Layer, Business Logic Layer, Data Access Layer) and technologies such as **HTML, CSS, JavaScript, PHP, and MySQL**, this project serves both as a real-world sales platform and a learning model for applying software engineering principles.

## 4. Project Requirements

### 4.1. Functional Requirements

- **User Management**
  - Register, login, update profile
  - Manage delivery addresses

- **Product Management**
  - Browse products
  - Search and view product details
  - Manage inventory (admin/staff)

- **Shopping Cart**
  - Add/remove products
  - Update quantities
  - Automatic total calculation

- **Order Processing**
  - Place orders
  - Select payment methods
  - Track order status (Processing, Shipping, Delivered, Canceled)

- **Payment System**
  - Cash on Delivery (COD)
  - VNPay online payment gateway

- **Admin Panel**
  - Manage users, products, categories, and orders
  - Role-based access control

- **Chatbot System**
  - Automated responses to common customer questions

- **Review System**
  - Product rating and reviews by verified customers
